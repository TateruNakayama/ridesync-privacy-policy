# RIDESYNC 子どもの安全基準

最終更新日: 2024-11-02

RIDESYNC はソーシャル／マッチングカテゴリ（グループライドのリアルタイム共有機能）に該当するため、Google Play の子どもの安全基準ポリシーに従い、公開済みの安全基準と連絡先を以下のとおり定義します。

## 1. 対象年齢と利用条件

- RIDESYNC は **13歳以上** を対象としたサービスです。
- 18歳未満のユーザーは、保護者または法定代理人の同意を得たうえで利用してください。
- 年齢詐称や不適切な利用が判明した場合は、アカウント停止や削除を行います。

## 2. コミュニティ／子どもの安全ガイドライン

RIDESYNC では以下の安全ルールを定め、アプリ内の「設定 > 安全について」で常時参照できるようにします。

1. **個人情報の保護**
   - 位置情報や居住地など、本人や第三者を特定できる情報を不用意に公開しないこと。
   - プロフィールにフルネームや連絡先を掲載しないこと。

2. **暴力的・不適切なコンテンツの禁止**
   - 暴力、差別、ハラスメント、露骨な表現、違法行為を示唆する投稿・画像・リンクを禁止します。

3. **安全な出会いとコミュニケーション**
   - RIDESYNC 上で知り合った相手と実際に会う場合は、保護者や信頼できる大人に必ず伝え、公共の場所を選ぶこと。
   - 18歳未満のユーザーに対する不適切な勧誘ややり取りは禁止し、即時通報の対象とします。

4. **報告・ブロック機能の利用推奨**
   - アプリ内の「プロフィール > ・・・ > 通報する」から不適切行為を報告できます。
   - ブロック機能で相手との接触を直ちに止められます。

5. **位置情報共有の最小化**
   - グループライドのフレンド以外とは共有されません

## 3. 監視と対応フロー

| 内容 | 具体的な対応 |
|------|--------------|
| 通報の受信 | アプリ内通報・メールで 24時間365日受け付け |
| 一次調査 | 24時間以内にモデレータがログと証跡を確認 |
| 対応 | 規約違反の場合は警告、一定期間の利用停止、もしくは即時アカウント削除 |
| 通知 | 通報者および関係者にメールまたはアプリ内通知で結果を共有 |
| 法執行対応 | 児童の安全に重大なリスクがある場合は、所轄の法執行機関へ通報 |

## 4. 保護者・監督者向け情報

- 子どもが RIDESYNC を利用する場合は、アプリ内のアクセス権限（位置情報、通知）を保護者が確認し、必要に応じて端末のペアレンタルコントロールを有効化してください。
- アカウント情報の削除は、アプリ内の「アカウント削除」もしくはサポート窓口に連絡することで実施できます。
